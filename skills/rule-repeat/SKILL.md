# Rule Repeat

Use this skill when the player asks to see Game Console rules again.

In an active game, repeat:

- the language-preference prompt
- how to enter actions
- the current scene/playbook options
- the freedom-within-framework rule: free-form actions are allowed, but the
  cartridge owns state, branch gates, consequences, and continuity

The slash command `/skill rule-repeat` returns the current game guide directly.
