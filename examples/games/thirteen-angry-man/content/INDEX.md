# Content Index

Use these handles with `game_lookup`:

- `case.md`: fixed case summary and deliberation boundary.
- `evidence.md`: admitted exhibits, testimony summaries, and critical nodes.
- `jurors.md`: juror profiles and switch gates.
- `room.md`: jury-room layout, heat, timing, and procedure.
- `endings.md`: ending eligibility notes.

All content is fixed background. Runtime vote, time, release, and trust state
must be read from the active save.
